<a name="english"></a>
## English

### What it does

This resource pack moves your scoreboard to the top right corner of the screen and adds a blue outline. Example below:  
<img src="score.png" width="1500">  

### Installation

1. Download `ScoreHud.zip`  
2. Place it in the `packs/` folder depending on your server software core. Personally, I use GeyserMC; for PocketMine-MP, add it to `resource_packs/` and then register it in `resource_packs.yml`  
3. Restart the server  

### Usage

If you want to make the scoreboard look like the one in the screenshot, add `\n` (line break) after each line! For example: `Balance: %eco_balance%\nOnline: %online%/100\n`

<a name="русский"></a>
## Русский

### Подробнее

Этот ресурс-пак перемещает ваш скорборд в правый верхний угол экрана и добавляет синюю обводку. Пример ниже:  
<img src="score.png" width="1500">  

### Установка

1. Скачайте `ScoreHud.zip`  
2. Поместите в папку `packs/` в зависимости от вашего ядра. Лично у меня GeyserMC, для PocketMine-MP добавьте в `resource_packs/` далее впишите его в `resource_packs.yml`  
3. Перезапустите сервер  

### Использование

Если хотите сделать скорборд как на скрине, то после каждой строчки ставьте "\n", то есть перенос строки! Например: `Баланс: %eco_balance%\nОнлайн: %online%/100\n`
